OpenMIA helps teams understand what their AI systems actually did at runtime. The first application module is **OpenMIA Observation**, a multi-tenant workspace for AI traces, spans, raw payloads, import jobs, feedback, and audit history.

Use these docs to set up a project, connect data sources, inspect traces, and keep operational evidence secure.

## What OpenMIA Observation gives you

- **Project isolation** for teams, products, environments, or customers.
- **Integration channels** for proxy capture, JSON/webhook ingestion, telemetry, and external platform imports.
- **Trace and span views** for model calls, tool use, runtime events, errors, latency, usage, and cost.
- **Raw-first ingestion** so every normalized trace can be traced back to its source payload.
- **Operational controls** for soft delete, restore, migration, audit, and export.
